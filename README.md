# Clasificacion Automatica de Neumonia en Radiografias de Torax

GitHub Page para el Trabajo 3 del curso Vision por Computador (3009228).

## Contenido

Este sitio web presenta los resultados del proyecto de clasificacion binaria de radiografias de torax (Normal vs Neumonia) utilizando:

- **Descriptores tradicionales**: HOG, LBP, GLCM, Momentos de Zernike
- **Modelos clasicos de ML**: SVM, Random Forest, XGBoost, k-NN
- **Deep Learning**: ResNet con Transfer Learning
